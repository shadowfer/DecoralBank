## 🎯 Sobre el Proyecto

DecoralBank es una aplicación bancaria moderna que combina funcionalidad avanzada con un diseño elegante y una experiencia de usuario excepcional. Desarrollada con React, TypeScript y Tailwind CSS para ofrecer rendimiento óptimo y una interfaz responsive.

- 🏦 **Interfaz Bancaria Moderna** - Diseño limpio y profesional
- 💳 **Gestión de Cuentas** - Administra múltiples cuentas bancarias
- 📊 **Historial de Transacciones** - Seguimiento detallado de movimientos
- 💰 **Vista de Balances** - Resúmenes en tiempo real
- 📱 **Diseño Responsive** - Optimizado para todos los dispositivos
- ⚡ **Alto Rendimiento** - Construido con Vite para velocidad máxima

## 🚀 Instalación y Configuración

### Prerrequisitos

Asegúrate de tener instalado:

- **Node.js** (v16 o superior)
- **npm** o **yarn**
- **Git**

### Pasos de Instalación

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/shadowfer/DecoralBank.git
   cd DecoralBank
   ```

2. **Instala las dependencias**
   ```bash
   npm install
   # o
   yarn install
   ```

3. **Inicia el servidor de desarrollo**
   ```bash
   npm run dev
   # o
   yarn dev
   ```

4. **¡Listo!** 🎉
   
   Abre tu navegador en `http://localhost:5173`

### Comandos Disponibles

```bash
npm run dev      # Servidor de desarrollo
npm run build    # Construir para producción
npm run preview  # Vista previa de producción
npm run lint     # Ejecutar ESLint
```
